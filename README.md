# personal-blog

vue-js web app built with [JSONPlaceholder](https://github.com/typicode/jsonplaceholder)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
